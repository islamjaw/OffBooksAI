# SyndicateAI 🎯

**Multi-agent fraud detection system with live adversarial testing**

A fraud detection system where an adversarial AI actively tries to evade detection in real-time, forcing the defense system to evolve.

## The Concept

Two AI agents battle in real-time:
- 🔴 **Red Team (FraudGPT):** Generates sophisticated fraud attacks
- 🔵 **Blue Team (DefenseAI):** Detects patterns and adapts

When Red wins → Blue learns and updates rules  
When Blue wins → Red tries smarter attacks  

**Result:** Self-improving fraud detection that finds blind spots before real criminals do.
